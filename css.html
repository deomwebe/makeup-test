CSS

CSS is the language we use to style an HTML document.And describes how HTML elements should be displayed.

how css can be inserted into the html page.

  • Inline CSS
  • Internal CSS
  • External CSS

CSS has alot of selectors and they include the following;

  1. Simple selectors (selects elements based on name, id, class)
  2. Combinator selectors (selects elements based on a specific relationship between them)
  3. Pseudo-class selectors (selects elements based on a certain state)
  4. Attribute selectors (selects elements based on an attribute or attribute values)
  5. Pseudo-elements selectors (selects and styles parts of an element)
